Contribution

For the simple problem of binary sum estimation, if I want to send 1 message per user only then the accuracy bounds by given by [2] and [1] are tight. You cannot do better! This paper proves a lower bound. If there is wiggle room on the number of messages I can send then the accuracy of the central case is achievable. They improve the bounds on the number of messages by making it disappear as \(n\) goes to infinity.
